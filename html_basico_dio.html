Explicação das Tags HTML

Tag <p>

A tag <p> é usada para criar um parágrafo no texto HTML. Ela define um bloco de texto que é separado por uma linha em branco acima e abaixo.

Tag <ol>

A tag <ol> é usada para criar uma lista ordenada. Cada item da lista é definido pela tag <li>, que deve ser colocada dentro da tag <ol>. Os itens da lista são numerados automaticamente.

  1. Item 1
  2. Item 2
  3. Item 3

Tag <h1> e <h2>

As tags <h1> e <h2> são usadas para definir cabeçalhos ou títulos na página. O <h1> é o título principal e normalmente é usado apenas uma vez na página, enquanto o <h2> é usado para títulos secundários ou subtítulos.
